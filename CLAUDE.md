## overview

- Lozza is a UCI chess engine written in C.

## toolchain

- clang based in a WSL2 Bash environment.

## status

- Early days, just got move gen. and PERFT working.

## nets

- `nets/weights.h` - Neural network weights, currently unused.

## build

- `make` - Creates ./lozza.

## code style

- 2 space indentation.
- return, continue and break on next line.
